Tooling for iRacing focused on telemetry, race planning and session management. Designed to improve race preparation and real-time visibility without unnecessary complexity.

**Source:** [github.com/cadfan/ira](https://github.com/cadfan/ira)

---

## Features

- Real-time telemetry display during sessions
- Session logging and data capture
- Background app launcher for race preparation
- Built in C with the Meson build system
